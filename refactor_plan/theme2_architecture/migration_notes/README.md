# Migration Notes

This directory contains notes and observations from the v2 provider migration process.

Each provider gets its own notes file (e.g., `openai_notes.md`, `cohere_notes.md`) that documents:

- Important decisions made during migration
- Deviations from the original plan
- Unexpected issues or blockers encountered
- Test results and any failing tests
- Performance observations
- API compatibility notes

These notes are created automatically by the `work_through_migration.sh` script as each provider is migrated.
